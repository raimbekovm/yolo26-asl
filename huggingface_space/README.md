# YOLO26 ASL Recognition

Real-time American Sign Language alphabet recognition using YOLO26-pose.

## Features

- **YOLO26-pose** - Latest Ultralytics model with NMS-free architecture
- **21 Hand Keypoints** - Precise detection using RLE
- **31 Classes** - 26 ASL letters + 5 gestures
- **Real-time** - 25+ FPS on CPU

## Usage

1. Upload an image with a hand showing an ASL letter
2. Or use the webcam for real-time recognition
3. View the predicted letter and confidence
